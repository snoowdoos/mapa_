🔄 Conceito Fundamental

Reações Reversíveis: A + B ⇌ C + D

Estado Dinâmico: Vdireta = Vinversa

Exemplo Clássico: N₂ + 3H₂ ⇌ 2NH₃

Sistema em constante movimento molecular

📊 Constante de Equilíbrio

Kc = [C]ᶜ[D]ᵈ / [A]ᵃ[B]ᵇ
Kp = Kc(RT)^Δn

K >> 1: Favorece produtos

K << 1: Favorece reagentes

K ≈ 1: Equilíbrio intermediário

🎯 Princípio de Le Chatelier

↑ Concentração: Sistema compensa

↑ Temperatura: Favorece endotérmica

↑ Pressão: Menor volume molar

Catalisador: Acelera ambos os sentidos

Resposta inteligente do sistema

🧮 Expressão Matemática

aA + bB ⇌ cC + dD

Incluir: Gases [atm] e aquosos [M]

Excluir: Sólidos e líquidos puros

Exemplo: CaCO₃(s) ⇌ CaO(s) + CO₂(g)

Atividade = 1 para fases condensadas

🏭 Aplicações Tecnológicas

Indústria Química:

• Processo Haber-Bosch (NH₃)

• Síntese do H₂SO₄ (contato)

Sistemas Biológicos:

• Transporte de O₂ (hemoglobina)

• Sistemas tampão (pH sanguíneo)

Essencial para a vida e indústria
